###文件说明
  - gruntfile 用于angular模板转换js和图片合并的

  - src下面的文件夹对应不同的项目

       - angular-html2js
            演示如何使用grunt-html2js
       - image-min
            演示图片压缩
       - angular-http
            则是重写了angular的部分http
       - angular-lazyload
            演示如何懒加载JS,CSS
       - angular-requirejs
            演示用requirejs管理小型项目时的项目组织
       - angular-requirejs-large
            演示用requirejs管理较大型项目时的项目组织
